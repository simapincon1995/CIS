This library is used to collect the Navex feedback ideas

Git Repo:
https://github.com/simapincon1995/CIS.git


Steps to install:

step-1: install the below library

npm i customer_idea_portal

step-2: Add script to body tag

 <script src="./node_modules/customer_idea_portal/script.js"></script>

 step-3: use web component element in html
    
<customer-idea-portal></customer-idea-portal>

codeSnippet:
<body>
    <!-- Custom Element: Customer Idea Portal -->
    <customer-idea-portal></customer-idea-portal>
    <script src="./node_modules/customer_idea_portal/script.js"></script>
</body>
